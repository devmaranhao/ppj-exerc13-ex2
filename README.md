### Qual padrão de projeto pode ser utilizado?
O padrão de projeto utilizado é o **Builder**, pois a classe Produto possui muitos atributos, alguns obrigatórios e outros opcionais, além de regras de dependência entre eles. O Builder facilita a criação do objeto, tornando o código mais legível e garantindo que o produto seja instanciado em um estado válido.
